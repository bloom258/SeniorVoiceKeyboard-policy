시니어 목소리 키보드 개인정보처리방침

💡 핵심 요약 및 보안 선언: 본 키보드 앱은 입력된 모든 텍스트를 외부 서버로 저장하거나 기록하지 않으며, 실시간 입력 내용 또한 별도로 수집되지 않습니다. 개인정보 보호를 위해 최신 보안 정책 및 운영체제(Android)의 가이드라인을 엄격히 준수합니다.

1. 개인정보의 처리 목적

  • 서비스 제공 및 기능 실행 (음성 인식 기반 텍스트 변환 등)
  • 가족 안심 안부 확인 서비스 제공 (마지막 활동 시간 공유)
  • 앱 안정성 확보 및 오류 분석
  • 기능 개선 및 품질 향상 (사투리 패턴 학습 및 사전 업데이트)
  • 광고 제공 및 성과 측정

2. 처리하는 데이터 및 수집 방법

  • 민감 입력 보호: 본 키보드는 비밀번호, 카드번호 등 민감한 입력 필드에서는 입력 내용을 수집하거나 기록하지 않으며, 운영체제(Android)의 보안 정책을 준수합니다.
  • 음성 데이터: 음성 인식 기능은 Google의 음성 인식 서비스(Google Speech-to-Text 등)를 통해 처리됩니다. 이 과정에서 음성 데이터가 외부 서버로 전송될 수 있으며, 해당 서비스 제공자의 개인정보처리방침이 적용됩니다.
  • 키보드 입력 데이터 (텍스트 보호): 일반적인 키보드 입력 내용은 기기 외부로 전송되지 않으며, 기기 내에서만 안전하게 처리됩니다. 기기 내에서만 처리되며 외부 서버로 전송되지 않습니다.
    ※ 앱 설정 시 나타나는 "개인 데이터 수집" 경고는 모든 서드파티 키보드에 공통으로 표시되는 안드로이드 시스템 메시지이며, 본 앱은 해당 경고와 달리 민감한 입력 내용(비밀번호, 카드번호 등)을 수집하지 않습니다.
  • 가족 안심 데이터 (선택 사항): 가족 안심 기능을 사용하는 경우, 자녀가 부모님의 안부를 확인할 수 있도록 마지막 키보드 사용 시간(타임스탬프)을 서버로 전송합니다. 해당 정보는 이용자의 명시적 동의 하에만 수집 및 공유되며, 어떠한 경우에도 입력된 대화 내용은 수집하거나 전송하지 않습니다.
  • 사투리 교정 데이터 (선택적 수집): 사용자가 동의한 경우에 한해, 교정한 사투리 조각이 수집될 수 있습니다. 이때 수집 범위는 사용자가 직접 입력한 전체 문장이 아닌, 일부 단어 또는 짧은 표현 단위로 제한되며, 개인 메시지, 연락처, 주소 등 개인 식별 가능 정보는 포함되지 않도록 처리됩니다.
  • 기술적 분석 정보 (자동 수집): 서비스 품질 개선을 위해 아래 항목이 익명화되어 수집될 수 있습니다:
    • 광고 ID(ADID), 앱 내부에서 생성된 익명 식별자(UUID), 기기 정보, 앱 오류 로그, 사용 패턴(비식별)

3. 제3자 서비스 이용 및 데이터 국외 이전

회사는 원활한 서비스 제공을 위해 아래 외부 서비스를 이용합니다. 광고 제공 등을 위해 제3자가 쿠키 또는 광고 식별자를 사용할 수 있으며, 수집되는 비식별 정보는 글로벌 서버로 이전되어 처리될 수 있습니다.

  • Google LLC (AdMob, Firebase Analytics / Crashlytics / Firestore, Google Speech-to-Text): Google 개인정보처리방침 확인하기
  • 데이터 국외 이전: 서비스 이용 과정에서 발생하는 익명화된 기술 정보 및 광고 식별자는 Google의 글로벌 서버(미국 등 해외)에 이전되어 처리될 수 있습니다.

4. 데이터 보유 및 파기 정책

  • 앱 삭제 시 기기 내 로컬 데이터는 즉시 파기됩니다.
  • 회원 탈퇴 또는 서비스 종료 시 수집된 활동 시간 정보 등은 즉시 파기됩니다.
  • 서비스 개선 목적의 익명 로그는 최대 14개월 동안 보존 후 파기됩니다.

5. 이용자의 권리 및 행사 방법

이용자는 기기 설정을 통해 언제든지 맞춤형 광고 수신을 제한하거나 식별자를 재설정할 수 있습니다.

  • 광고 ID 설정: 스마트폰 설정 -> 개인정보보호 -> 광고 -> 광고 ID 재설정 또는 삭제
  • 데이터 거부: 앱 내 설정 화면을 통해 선택적 데이터 제공 동의를 언제든 철회할 수 있습니다.

단, 일부 기능은 데이터 제공에 동의하지 않을 경우 정상적으로 작동하지 않을 수 있습니다.

6. 개인정보처리방침의 변경

본 방침은 관련 법령 또는 서비스 정책 변경에 따라 수정될 수 있으며, 변경 시 앱 내 공지사항 또는 연결 페이지를 통해 지체 없이 고지합니다.
