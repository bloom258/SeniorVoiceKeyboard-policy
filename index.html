시니어 목소리 키보드 개인정보처리방침

💡 핵심 요약 및 보안 선언:

본 키보드 앱은 사용자의 일반적인 입력 텍스트 및 대화 내용을 외부 서버에 저장하지 않습니다.
단, 음성 인식 기능 사용 시에는 Google의 음성 인식 서비스로 데이터가 전송될 수 있으며, 해당 데이터는 당사 서버에 저장되지 않습니다.
개인정보 보호를 위해 안드로이드(Android) 운영체제의 보안 가이드라인을 엄격히 준수하며, 서비스 운영 및 품질 개선에 필요한 최소한의 비식별 데이터만 안전하게 처리합니다.

1. 개인정보의 처리 목적

  • 음성 인식 기반 텍스트 입력 기능 제공
  • 가족 안심 기능 제공 (선택 시 마지막 사용 시간 공유)
  • 앱 안정성 확보, 오류 분석 및 건전한 서비스 환경 유지
  • 사투리 사전 품질 개선 및 서비스 업데이트
  • 광고 제공 및 성과 측정

2. 처리하는 데이터 및 수집 방법

  • 일반 텍스트 입력 철저 보호: 본 키보드는 일상적인 대화, 비밀번호, 계좌번호 등 일반 및 민감 입력 내용을 당사 서버로 전송하거나 저장하지 않습니다.
  • 음성 데이터: 음성 인식 기능은 Google의 음성 인식 서비스(Google Speech-to-Text)를 통해 처리되며, 이 과정에서 음성 데이터가 해당 서비스로 전송될 수 있습니다. 해당 데이터는 당사 서버에 저장되지 않으며, Google의 개인정보처리방침을 따릅니다.
  • 어뷰징 방지 및 악성 사용자 제한 (예외): 서비스 안정성을 심각하게 저해하는 행위(예: 욕설 등 금칙어의 반복 입력 패턴)가 탐지되는 경우에 한해, 쾌적한 서비스 환경 유지를 위해 입력된 문장의 원문이 아닌, 제한된 패턴 정보 및 비식별화된 식별자가 서버로 전송되어 처리될 수 있습니다.
  • 사투리 제보 및 서비스 개선 (선택적 수집): 사용자가 '사투리 제보' 등 데이터 제공 기능에 명시적으로 참여하여 동의한 경우에 한해, 교정한 결과의 일부가 비식별 형태로 수집되어 사투리 사전 품질 개선에 활용됩니다. 개인 식별 정보는 일절 포함되지 않습니다.
  • 가족 안심 데이터 (선택): 가족 안심 기능을 켜신 경우에 한해, 자녀가 안부를 확인할 수 있도록 '마지막 키보드 사용 시간(타임스탬프)'만을 서버로 전송합니다. 어떠한 경우에도 입력된 대화 내용은 전송하지 않습니다.
  • 기술적 분석 정보 (자동 수집): 앱 오류 로그, 비식별 기기 정보, 광고 ID(ADID) 등이 앱 안정성 확보 및 광고 제공을 위해 자동 수집될 수 있습니다.

3. 제3자 서비스 이용 및 데이터 국외 이전

회사는 안정적인 서비스 제공을 위해 아래의 외부 서비스를 이용하며, 비식별화된 기술 정보 및 광고 식별자는 글로벌 서버(미국 등)로 이전되어 처리될 수 있습니다.

  • Google LLC (AdMob, Firebase Analytics / Crashlytics / Firestore, Google Speech-to-Text)

4. 계정 삭제 및 데이터 파기 정책

  • 사용자는 언제든지 [앱 내 설정 → 회원 탈퇴 및 데이터 초기화] 메뉴를 통해 계정을 삭제할 수 있습니다.
  • 탈퇴 시 클라우드 서버에 저장된 아래 데이터는 즉시 영구 삭제되며 복구할 수 없습니다:
    • Firebase Auth 익명 계정 정보
    • Firestore 사용자 데이터 (users 컬렉션)
    • 기기 식별 기반 체험판 데이터 (trial_users 컬렉션)
  • 주의사항: 단순히 기기에서 앱을 삭제하는 것만으로는 서버에 보관된 익명 식별자 데이터가 즉시 파기되지 않을 수 있으므로, 완전한 파기를 원하실 경우 반드시 앱 내 '회원 탈퇴' 기능을 이용해 주시기 바랍니다.
  • 서비스 개선 및 안정성 목적의 익명 로그(어뷰징 제한 기록 포함)는 최대 14개월 보존 후 자동 파기됩니다.

5. 이용자의 권리 및 행사 방법

이용자는 스마트폰 설정을 통해 언제든지 맞춤형 광고 수신을 제한하거나, 앱 내 설정을 통해 데이터 제공을 철회할 수 있습니다.

  • 광고 ID 재설정: 스마트폰 설정 → 개인정보보호 → 광고 → 광고 ID 재설정 또는 삭제
  • 데이터 거부: 앱 내 설정 화면을 통해 선택적 데이터 수집 동의를 해제할 수 있습니다. (단, 관련 기능 이용이 제한될 수 있습니다.)
